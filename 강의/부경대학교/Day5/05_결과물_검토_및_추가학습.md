# 05. 결과물 검토 및 추가 학습 자료

## 코드 리뷰

### 코드 검토 받고 싶은 사람?

**손 들어주세요!**

- 작성한 코드를 제가 검토해드릴 수 있습니다
- **PR(Pull Request) 리뷰 받아보고 싶은 사람**도 손 들어주세요

> 💡 **왜 PR을 받아야 할까?**
>
> **PR(코드 리뷰)을 받아야 성장합니다.**
>
> - 혼자 코드를 짜면 본인의 방식만 알게 됩니다
> - 다른 사람의 피드백을 받으면 **더 나은 방법**을 배울 수 있습니다
> - 실무에서는 모든 코드가 PR을 거쳐서 배포됩니다
> - **PR 리뷰 과정**에서 많이 배우고 성장합니다
>
> 지금이 아니면 PR 리뷰 받아볼 기회가 많지 않으니, 받고 싶은 분은 편하게 요청하세요!

### 좋은 코드 예시

**잘 작성된 코드의 특징**:

1. **명확한 함수명**: `extract_data()`, `transform_data()`, `load_data()`
2. **로깅**: 각 단계마다 진행 상황 기록
3. **에러 처리**: `try-except`로 예외 처리
4. **주석**: 복잡한 로직에 설명 추가
5. **모듈화**: 재사용 가능한 함수로 분리

### 개선 가능한 부분

**더 나은 코드를 위한 제안**:

1. **설정 분리**: 경로, 파라미터를 설정 파일로 분리 (config.yaml)
2. **테스트 추가**: 단위 테스트 작성 (pytest)
3. **타입 힌트**: 함수 파라미터와 반환값에 타입 명시
4. **문서화**: Docstring 추가

## 추가 학습 자료 안내

### 추천 학습 경로

#### 1. 기초 다지기

**Linux 고급 명령어**

- `awk`, `sed`: 텍스트 처리
- Shell 스크립팅: 자동화

**Python 심화**

- OOP (객체 지향 프로그래밍)
- 비동기 프로그래밍 (`async`/`await`)

**SQL 마스터하기**

- 윈도우 함수 (`ROW_NUMBER`, `RANK`, `LAG`, `LEAD`)
- CTE (Common Table Expressions)
- 쿼리 최적화

---

#### 2. 데이터 엔지니어링 심화

**Apache Kafka**

- 실시간 데이터 스트리밍
- Producer, Consumer, Topic
- 추천 자료: "Kafka: The Definitive Guide" (O'Reilly)

**Apache Airflow**

- 워크플로우 관리
- DAG (Directed Acyclic Graph) 작성
- 스케줄링, 모니터링
- 추천 자료: Airflow 공식 문서

**dbt (data build tool)**

- SQL 기반 데이터 변환
- 모델링, 테스트, 문서화
- 추천 자료: dbt 공식 문서, dbt Learn

**Delta Lake**

- 데이터 레이크 관리
- ACID 트랜잭션, 타임 트래블
- 추천 자료: Delta Lake 공식 문서

---

#### 3. 클라우드 플랫폼

**AWS**

- **S3**: 객체 스토리지
- **EMR**: Spark 클러스터 관리
- **Glue**: ETL 서비스
- **Redshift**: 데이터 웨어하우스
- 추천 자료: AWS 공식 문서

**GCP**

- **BigQuery**: 서버리스 데이터 웨어하우스
- **Dataflow**: Apache Beam 기반 스트리밍/배치 처리
- **Dataproc**: Spark 클러스터 관리
- 추천 자료: GCP 공식 문서

**Azure**

- **Data Factory**: ETL 서비스
- **Synapse Analytics**: 데이터 웨어하우스
- **Databricks**: Spark 기반 분석 플랫폼
- 추천 자료: Azure 공식 문서

---

#### 4. 실전 프로젝트

**개인 프로젝트 아이디어**
